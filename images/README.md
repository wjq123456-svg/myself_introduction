# 图片文件夹说明

## 📁 文件结构
```
images/
├── profile.jpg      # 首页个人照片 (建议 400x400 像素)
├── avatar.jpg       # 个人信息部分头像 (建议 300x400 像素)
├── work1.jpg        # 工作照片1 (建议 300x300 像素)
├── work2.jpg        # 工作照片2 (建议 300x300 像素)
├── life1.jpg        # 生活照片1 (建议 300x300 像素)
├── life2.jpg        # 生活照片2 (建议 300x300 像素)
├── travel1.jpg      # 旅行照片1 (建议 300x300 像素)
└── travel2.jpg      # 旅行照片2 (建议 300x300 像素)
```

## 📸 如何添加本地照片

### 步骤1：准备照片
1. 选择你的个人照片和生活照片
2. 建议使用 JPG 或 PNG 格式
3. 可以使用图片编辑软件调整尺寸

### 步骤2：重命名文件
将你的照片按照上面的文件名重命名：
- `profile.jpg` - 首页显示的主要个人照片
- `avatar.jpg` - 个人信息部分的照片
- `work1.jpg`, `work2.jpg` - 工作相关照片
- `life1.jpg`, `life2.jpg` - 生活照片
- `travel1.jpg`, `travel2.jpg` - 旅行照片

### 步骤3：放置照片
将重命名后的照片文件复制到 `images` 文件夹中，替换现有的占位文件。

### 步骤4：查看效果
在浏览器中打开 `personal-profile.html` 文件，查看照片显示效果。

## 🎨 照片建议

### 首页照片 (profile.jpg)
- 尺寸：400x400 像素（正方形）
- 内容：清晰的个人正面照
- 背景：简洁的背景
- 格式：JPG（文件较小）或 PNG（质量更高）

### 个人信息照片 (avatar.jpg)
- 尺寸：300x400 像素（竖版）
- 内容：半身照或全身照
- 风格：专业、友好

### 画廊照片
- 尺寸：300x300 像素（正方形）
- 内容：工作、生活、旅行相关
- 风格：真实、自然

## 📱 图片优化建议

1. **压缩图片**：使用在线工具（如 TinyPNG）压缩图片，减少加载时间
2. **统一风格**：保持照片色调和风格的一致性
3. **高质量**：确保照片清晰度，避免模糊
4. **版权**：使用自己拍摄或有使用权的照片

## 🔧 如果照片不显示

1. 检查文件名是否正确
2. 确保照片文件确实在 `images` 文件夹中
3. 检查文件扩展名（.jpg 或 .png）
4. 清除浏览器缓存后重新加载

## 📝 添加更多照片

如果你想添加更多照片，可以：
1. 将新照片放入 `images` 文件夹
2. 在 HTML 文件中添加新的 `<img>` 标签
3. 确保路径正确：`images/你的照片名.jpg`

## 🌟 高级技巧

- 可以使用 WebP 格式进一步优化加载速度
- 考虑为不同设备准备不同尺寸的照片
- 使用 CSS 添加照片滤镜效果
